## Anonymous WEB
- Web, Android app을 통한 접근

### 사용 기술
- 서버
    - Node.js, mongo-db
- 클라이언트
    - web : react, html5, css3
    - android : react-native
- 배포
    - heroku

### 기능
1. 회원 가입
    - 개인정보 암호화
2. 개인 페이지
    - 개인 정보 변경
        - 비밀번호 변경
        - 닉네임 변경
    - 작성 글 모아보기
3. 글 
    - 작성
    - 수정
    - 삭제
4. 좋아요
5. 댓글 포스트 (트위터 참고)
    - 연결되는 origin 포스트는 1개
    - 참조되는 것은 여러개 가능 (리스트 형식)
